In-progress project for the 100daydash.blog challenge, focused on turning NHTSA public recall API data into a repeatable vehicle safety analytics platform.

<!--more-->

The project is scoped around practical data work: ingesting recall records from the public API, shaping them into medallion-style analytical tables, and preparing relational, dashboard-ready outputs for trend, manufacturer, component, and campaign-level reporting.

Current focus areas:

- NHTSA public recall API ingestion
- Repeatable data cleaning and normalization
- Modeled recall and campaign analytics
- PostgreSQL/Supabase-ready data structures
- Dashboard-ready outputs for Power BI analysis
- Documentation of design decisions during the 100daydash.blog challenge
